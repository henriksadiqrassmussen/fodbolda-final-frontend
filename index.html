Sådan bruges appen

  1. Start med Ligaer-siden og tilføj de ligaer du vil følge.
  2. Brug Admin/Dashboard til at hente data og 5 års historik.
  3. Brug Arkiv, Indbyrdes og Oddssætter som hver sin rene arbejdsflade.
  4. Railway backend kører med npm run backend og PostgreSQL via DATABASE_URL.
